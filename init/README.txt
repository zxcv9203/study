#                            git 초기화                            #
#                                                                #
#                                                                #
#                                                                #
#                                                                #
##################################################################

1. 해당 디렉터리에 git을 사용하기 위해 git init이라는 명령을 이용해 초기화를 한다.

2. 명령어를 실행하면 해당 디렉터리에 .git 이라는 폴더가 생성된다. 이 .git을 로컬 저장소라고 부른다.

3. 로컬 저장소에 내가 만든 버전 정보, 원격 저장소 주소 등이 저장된다.

4. git clone 등을 이용하여 원격 저장소에 받아올 경우 로컬 저장소가 자동으로 생성된다.

*주의* 한 디렉터리에 하나의 로컬 저장소만 존재해야 합니다.
